# YugiohLegend 기획서
업데이트: 2026-06-08

## 개요
YugiohLegend는 1600 x 900 가로 화면에서 진행되는 3라인 자동 전투 카드 게임이다. 플레이어는 손패에서 몬스터와 마법을 라인에 배치하고, 공개 후 각 라인에서 자동 전투를 해결한다.

## 핵심 규칙
- 시작 LP는 4000이다.
- 시작 손패는 4장이며 매 턴 1장을 드로우한다.
- 총 4턴으로 진행한다.
- 1턴은 배치 전용 턴이며 공격하지 않는다.
- 2턴부터 공개 후 라인별 자동 전투가 발생한다.
- 라인은 총 3개를 사용한다.
- 1턴은 1라인, 2턴은 2라인, 3턴부터는 3라인을 사용한다.
- 4턴 종료 후 최종 전투까지 해결하고 LP가 높은 쪽이 승리한다.

## 전투 규칙
- 같은 라인에 양쪽 몬스터가 있으면 전투력이 낮은 몬스터가 전투력 차이만큼 HP를 잃는다.
- HP가 정확히 0이 되면 몬스터는 사망하고 필드에서 제거된다.
- HP가 0 아래로 내려가면 몬스터는 사망하고, 초과 피해만큼 해당 플레이어 LP가 감소한다.
- 같은 라인에 상대 몬스터가 없으면 공격 몬스터의 전투력만큼 상대 LP를 직접 공격한다.
- 같은 전투력으로 맞붙으면 HP와 LP 피해가 발생하지 않는다.
- 전투 로그는 몬스터전 피해를 `HP -수치`, 관통 피해가 있으면 `HP -수치 / LP -수치`로 표시한다.

## 카드 구분
| 구분 | 설명 |
| --- | --- |
| FREE | 제물 없이 일반 소환 가능한 몬스터 |
| TRIBUTE | 이미 필드에 소환된 아군 몬스터를 제물로 바쳐 소환하는 몬스터 |
| STRIKER | 전투력이 높고 생명력이 낮은 기본 몬스터 |
| GUARDIAN | 생명력이 높고 전투력이 낮은 기본 몬스터 |
| UTILITY | 전투력과 생명력은 낮지만 이동 또는 특수 효과가 있는 기본 몬스터 |
| BRUISER | 강한 전투력과 생명력을 가진 제물 몬스터 |
| ALLY+ | 아군을 강화하는 제물 몬스터 |
| FIELD+ | 라인 또는 필드 상태를 강화하는 제물 몬스터 |
| MOBILE | 제물이 필요하지만 이동성이 있는 몬스터 |
| SCALE | 바친 제물 수만큼 강해지는 몬스터 |

## 제물 소환
- 제물은 손패 카드가 아니라 이미 필드에 소환된 아군 몬스터만 사용할 수 있다.
- 제물 몬스터를 필드에 내려놓으면 제물 선택 플로우가 시작된다.
- 필요한 수만큼 필드 위 아군 몬스터를 직접 선택해야 `COMMIT`이 활성화된다.
- 제물로 선택된 몬스터는 소환 해결 시 필드에서 제거된다.

## 마법 카드
- 함정은 별도 카드 종류로 쓰지 않고, 앞면 마법과 뒷면 마법으로 통합한다.
- 앞면 마법은 라인에 공개 배치되고 몇 턴 후 효과가 발동한다.
- 앞면 마법이 발동 전에 파괴되면 효과는 사라진다.
- 뒷면 마법은 세트된 상태로 있다가 조건이 달성되면 공개되며 효과를 발동한다.
- 주요 효과는 전투력 상승, 상대 몬스터 제거, 상대 마법 제거, 공격 무효화, 직접 피해 감소다.
- `Second Gate`는 앞면 마법이며, 살아남아 다음 턴에 발동하면 그 턴 소환 가능 횟수를 1회 추가한다.
- `Overextend Collapse`는 뒷면 마법이며, 상대가 이번 턴에 몬스터를 2회 이상 성공적으로 소환하면 발동해 필드 위 모든 몬스터를 제거한다.

## 덱 편성
- 덱은 정확히 12장이어야 저장 및 결투 시작이 가능하다.
- 덱 편성 화면은 카드 아카이브와 12개의 덱 슬롯으로 구성한다.
- 카드는 클릭 또는 드래그로 슬롯에 넣을 수 있다.
- 슬롯에 들어간 카드는 카드 미니 프리뷰로 보이고, 클릭하면 제거된다.
- `AUTO FILL`은 초보자용 추천 덱을 자동 구성한다.
- `SAVE DECK`은 유효한 12장 덱만 저장한다.
- `SAVE AND DUEL`은 저장 후 즉시 결투를 시작한다.

## UI/UX
- 화면은 가로형 1600 x 900 레이아웃이다.
- 손패, 3라인 전장, 상대/플레이어 HUD, 덱 카운터, 상태 가이드, 커밋 버튼을 한 화면에서 확인할 수 있다.
- 소환 또는 배치 가능한 카드는 금색 글로우로 표시한다.
- 카드에는 종류, 소환 방식, ATK/HP, 제물 필요 여부를 명확히 표시한다.
- 제물 소환 후보 라인은 전용 하이라이트로 안내한다.
- 현재 버전은 마나 비용과 마나 게이지를 사용하지 않는다.

## 아트 적용
- 현재 카드 구성 18종에 맞춘 카드 일러스트를 적용했다.
- 카드별 이미지 경로는 `client/public/assets/cards/{cardId}.png`이다.
- 버튼, HUD, 프레임, 손패 영역, 라인 프레임에는 게임 전용 텍스처를 사용한다.
