# YugiohLegend 기획서

업데이트: 2026-06-08

## 개요
YugiohLegend는 1600 x 900 가로 화면에서 진행되는 3라인 자동 전투 카드 게임이다. 플레이어는 덱에서 뽑은 몬스터와 마법을 라인에 배치하고, 공개 후 각 라인에서 자동 전투를 해결한다.

## 핵심 규칙
- 시작 LP는 4000이다.
- 시작 손패는 4장이며 매 턴 1장을 드로우한다.
- 총 4턴으로 진행한다.
- 1턴은 배치 전용 턴이며 공격하지 않는다.
- 2턴부터 공개 후 라인별 자동 전투가 발생한다.
- 4턴 종료 후 LP가 높은 쪽이 승리한다.
- 라인은 총 3개를 사용한다.

## 라인 해금
| 턴 | 사용 가능한 라인 |
| --- | --- |
| 1턴 | 중앙 1라인 |
| 2턴 | 좌측 + 중앙 2라인 |
| 3턴 | 좌측 + 중앙 + 우측 3라인 |
| 4턴 | 전체 3라인 |

잠긴 라인은 소환과 마법 배치를 막는다. 클라이언트는 잠긴 라인을 어둡게 표시하고 입력을 차단한다.

## 카드 구분
몬스터 카드는 전투력과 생명력을 하스스톤식으로 동시에 가진다.

| 구분 | 설명 |
| --- | --- |
| FREE | 제물 없이 일반 소환 가능한 몬스터 |
| TRIBUTE | 이미 필드에 소환된 아군 몬스터를 바쳐야 소환 가능한 몬스터 |
| STRIKER | 전투력이 높고 생명력이 낮은 기본 몬스터 |
| GUARDIAN | 생명력이 높고 전투력이 낮은 기본 몬스터 |
| UTILITY | 전투력과 생명력은 낮지만 이동 또는 특수효과가 있는 기본 몬스터 |
| BRUISER | 강한 전투력과 생명력을 가진 제물 몬스터 |
| ALLY+ | 아군 몬스터를 강화하는 제물 몬스터 |
| FIELD+ | 라인 또는 필드 상태를 강화하는 제물 몬스터 |
| MOBILE | 제물이 필요하지만 라인 이동성이 있는 제물 몬스터 |
| SCALE | 바친 제물 수만큼 강해지는 제물 몬스터 |

제물 소환은 손패 카드가 아니라 이미 필드에 소환된 아군 몬스터만 바칠 수 있다. 플레이어가 제물 라인을 직접 지정하지 않아도 서버가 소환 라인을 제외한 아군 필드 몬스터 중 필요한 수만큼 자동 선택한다.

## 마법 카드
함정 카드는 별도 종류로 두지 않고 앞면 마법과 뒷면 마법으로 통합한다.

| 구분 | 동작 |
| --- | --- |
| 앞면 마법 | 라인에 공개 배치되며 몇 턴 뒤 효과가 발동한다. 발동 전에 제거되면 효과가 사라진다. |
| 뒷면 마법 | 뒤집어 배치하고 조건이 달성되면 공개되며 강한 효과를 발동한다. |

마법은 라인의 규칙을 바꾸는 느낌을 중심으로 구성한다. 주요 효과는 전투력 상승, 상대 몬스터 제거, 상대 마법 제거다.

## 전투 규칙
- 같은 라인에 양측 몬스터가 있으면 전투력 차이만큼 약한 쪽 몬스터의 생명력을 잃는다.
- 생명력이 0 이하가 된 몬스터는 파괴된다.
- 상대 라인이 비어 있으면 몬스터가 상대 LP에 직접 피해를 준다.
- 1턴은 공격하지 않는 배치 턴이다.

## 덱 구성 과정
덱 구성은 별도 Deck Builder 화면에서 진행한다.

1. 카드 아카이브에서 원하는 카드를 선택한다.
2. 덱은 8장 이상 12장 이하로 구성한다.
3. 현재 덱은 몬스터, 기본 몬스터, 제물 몬스터, 앞면 마법, 뒷면 마법 수를 즉시 보여준다.
4. 현재 덱은 텍스트 목록이 아니라 12개의 슬롯으로 표시한다.
5. 카드 아카이브의 카드는 클릭으로 첫 빈 슬롯에 넣거나, 드래그해서 원하는 빈 슬롯에 직접 넣을 수 있다.
6. 슬롯에 들어간 카드는 카드 미니 프리뷰로 표시하고, 클릭하면 제거된다.
7. `AUTO FILL`은 초보자용 추천 덱을 슬롯에 자동 구성한다.
8. `SAVE DECK`은 저장만 수행한다.
9. `SAVE AND DUEL`은 저장 후 즉시 대전을 시작한다.
10. 메인 메뉴의 `SOLO DUEL`은 유효한 저장 덱이 있으면 그 덱으로 시작하고, 없으면 Deck Builder로 이동한다.

추천 초보 덱은 기본 몬스터, 1장 이상의 제물 몬스터, 라인 제어 마법을 섞어 제물 소환과 마법 타이밍을 자연스럽게 경험하도록 구성한다.

## 최신 UI 구성
- 기본 해상도는 1600 x 900이다.
- 가로형 전장 레이아웃을 사용한다.
- 손패, 3라인 전장, 상대/플레이어 HUD, 커밋 버튼을 한 화면에서 확인할 수 있다.
- 소환 또는 배치 가능한 카드는 녹색 글로우로 표시한다.
- 카드에는 종류, 사용 방식, ATK/HP, 제물 필요 여부를 한눈에 보이게 표시한다.
- 제물 소환 후보 라인은 `TRIBUTE` 오버레이로 안내한다.
- 버튼, HUD, 손패 보관대, 라인 프레임은 전용 텍스처를 사용한다.
- 상단 가이드 HUD는 짧은 2줄 안내로 고정해 프레임 밖으로 문구가 잘리지 않게 한다.
- 필드에 소환된 말은 라인 프레임보다 아주 살짝 크게 표시해 판 밖으로 조금 돌출되는 연출을 준다.
- 현재 버전에는 마나 비용/마나 게이지 개념을 사용하지 않는다.

## 카드 아트 적용
- 현재 카드 구성 18종에 맞춘 AI 카드 일러스트를 적용했다.
- 원본 아트 아틀라스: `client/public/assets/cards/card_art_atlas.png`
- 카드별 이미지 경로: `client/public/assets/cards/{cardId}.png`
- `BootScene`에서 모든 카드 이미지를 로드하고 `CardSprite`가 카드 ID에 맞는 이미지를 표시한다.

## UX 보강 방향
- 첫 플레이어가 즉시 이해할 수 있도록 온보딩 화면을 제공한다.
- 플레이 가능한 카드 글로우, 턴 단계 표시, 남은 턴 수 표시를 제공한다.
- 사용할 수 없는 카드와 라인은 흐리게 표시하고 상태 메시지로 이유를 안내한다.
- 카드 선택 시 실제 배치 가능한 라인만 강조한다.
- 덱빌더에서는 저장 가능한 덱인지 즉시 피드백한다.
