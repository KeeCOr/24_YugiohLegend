# YugiohLegend 기획서
업데이트: 2026-05-28

## 개요
YugiohLegend는 1600 x 900 가로 화면에서 진행되는 3라인 자동 전투 카드 게임이다. 플레이어는 매 턴 손패에서 몬스터 또는 마법을 골라 라인에 배치하고, 양측 행동 공개 후 각 라인에서 전투를 해결한다.

## 핵심 규칙
- 시작 LP는 4000이다.
- 시작 손패는 4장이고 매 턴 1장을 드로우한다.
- 총 4턴으로 진행한다.
- 1턴은 배치 전용 턴이며 공격하지 않는다.
- 2턴부터 공개 후 자동 전투가 발생한다.
- 4턴 종료 후 LP가 높은 쪽이 승리한다.
- 라인은 총 3개이며 턴에 따라 순차 해금된다.

## 라인 해금
| 턴 | 사용 가능한 라인 |
| --- | --- |
| 1턴 | 중앙 1라인 |
| 2턴 | 왼쪽 + 중앙 2라인 |
| 3턴 | 왼쪽 + 중앙 + 오른쪽 3라인 |
| 4턴 | 전체 3라인 |

잠긴 라인은 소환과 마법 배치를 무시한다. 클라이언트는 잠긴 라인을 어둡게 표시하고 클릭 입력을 차단한다.

## 카드 구분
몬스터 카드는 하스스톤처럼 전투력과 생명력을 동시에 가진다.

| 구분 | 설명 |
| --- | --- |
| FREE | 제물 없이 일반 소환 가능 |
| TRIBUTE | 제물이 필요한 상급 소환 |
| STRIKER | 전투력은 높지만 생명력이 낮은 기본 몬스터 |
| GUARDIAN | 생명력은 높지만 전투력이 낮은 기본 몬스터 |
| UTILITY | 능력치가 낮지만 이동 또는 특수효과가 있는 기본 몬스터 |
| BRUISER | 제물을 요구하며 전투력과 생명력이 모두 높은 몬스터 |
| ALLY+ | 아군 몬스터를 강화하는 상급 몬스터 |
| FIELD+ | 다른 라인 또는 필드 상태를 강화하는 상급 몬스터 |
| MOBILE | 제물을 요구하지만 라인 이동성이 있는 상급 몬스터 |
| SCALE | 제물 수만큼 강해지는 상급 몬스터 |

카드 상단에는 `MONSTER - FREE`, `MONSTER - TRIBUTE`, `SPELL - FACE`, `SPELL - SET`처럼 종류와 사용 방식을 표시한다. 손패에서도 ATK/HP, 소환 등급, 카드 종류가 바로 읽히도록 카드 크기와 정보 배치를 키웠다.

## 전투 규칙
- 같은 라인에 양측 몬스터가 있으면 전투력 차이만큼 약한 쪽 몬스터의 생명력을 잃는다.
- 생명력이 0 이하가 된 몬스터는 파괴된다.
- 상대 라인이 비어 있으면 몬스터가 상대 LP에 직접 피해를 준다.
- 1턴은 공격하지 않는 배치 턴으로 처리한다.

## 마법 카드
마법은 즉발 카드가 아니라 라인의 규칙을 바꾸는 장치로 설계한다.

| 구분 | 동작 |
| --- | --- |
| 앞면 마법 | 라인에 공개 배치되며 몇 턴 뒤 효과가 발동한다. 발동 전에 제거되면 효과가 사라진다. |
| 뒷면 마법 | 뒤집어 배치하고 조건이 달성되면 공개되며 효과가 발동한다. |

주요 효과는 전투력 상승, 상대 몬스터 제거, 상대 마법 제거 중심이다. 함정 카드는 별도 종류로 두지 않고 뒷면 마법으로 통합한다.

## 최신 UI 구성
- 기준 해상도는 1600 x 900이다.
- 왼쪽에는 손패, 중앙에는 3라인 전장, 오른쪽에는 상대/플레이어 HUD와 커밋 버튼을 배치한다.
- 손패는 세로 팬 형태로 펼쳐 카드가 겹쳐 안 보이는 문제를 줄였다.
- 소환 또는 배치 가능한 카드는 녹색 글로우로 표시한다.
- 전장 상단에는 게임명, 현재 턴, 진행 단계, 간단한 입력 가이드를 표시한다.
- 오른쪽 HUD에는 생성된 카드 이미지를 사용한 상대/플레이어 초상, LP, 덱 카운터를 표시한다.
- 라인에 놓인 앞면/뒷면 마법도 해당 카드 이미지를 작은 아이콘으로 표시한다.

## 카드 아트 적용
- 현재 카드 구성 18종에 맞춰 AI 카드 일러스트를 생성했다.
- 원본 아틀라스: `client/public/assets/cards/card_art_atlas.png`
- 카드별 경로: `client/public/assets/cards/{cardId}.png`
- `BootScene`에서 모든 카드 이미지를 로드하고 `CardSprite`가 카드 ID에 맞는 이미지를 표시한다.
- 주요 HUD 초상과 마법 슬롯 아이콘에도 같은 카드 이미지를 재사용한다.

## UX 보강 방향
- 첫 플레이어가 즉시 이해할 수 있도록 플레이 가능한 카드 글로우, 턴/단계 표시, 덱 수량 표시를 제공한다.
- 손패 선택 후 라인을 누르는 흐름을 상단 가이드와 상태 메시지로 반복 안내한다.
- 카드 종류, 소환 가능 여부, 전투력/생명력은 손패 상태에서도 항상 노출한다.
