YugiohLegend 기획서

3라인 자동 전투 카드 게임. 플레이어는 매 턴 카드를 배치하고, 공개 단계 이후 각 라인에서 전투를 해결한다.

업데이트: 2026-05-19 해상도: 1600 x 900 라인: 3개 최신 실행파일 갱신 대상

핵심 규칙

  • 시작 LP는 4000이다.
  • 시작 손패는 4장이고, 매 턴 1장을 드로우한다.
  • 총 4턴으로 진행하며 1턴은 배치 전용 턴이다.
  • 2턴부터 공개 후 자동 전투가 발생한다.
  • 각 플레이어는 3개의 라인을 가진다.
  • 각 라인에는 몬스터 1장과 마법 카드가 배치될 수 있다.

라인 해금

턴사용 가능 라인
1턴중앙 라인
2턴왼쪽 + 중앙 라인
3턴왼쪽 + 중앙 + 오른쪽 라인
4턴3개 라인 전체

잠긴 라인은 서버에서 소환과 마법 배치를 무시하고, 클라이언트는 잠긴 상태를 화면에 표시한다.

카드 구분

FREE
제물 없이 일반 소환 가능.
TRIBUTE
제물이 필요한 상급 소환.
STRIKER
전투력은 높지만 생명력이 낮은 기본 몬스터.
GUARDIAN
생명력은 높지만 전투력이 낮은 기본 몬스터.
UTILITY
전투력과 생명력은 낮지만 이동 또는 특수효과를 가진 기본 몬스터.
BRUISER / ALLY+ / FIELD+
제물을 요구하며 강한 전투, 아군 강화, 필드 강화 역할을 맡는 몬스터.

카드 상단에는 MONSTER - FREE, MONSTER - TRIBUTE, SPELL - FACE, SPELL - SET처럼 종류와 사용 방식을 표시한다.

전투 규칙

  • 같은 라인에 양쪽 몬스터가 있으면 전투력 차이만큼 패배한 쪽 몬스터의 생명력을 잃는다.
  • 생명력이 0 이하가 된 몬스터는 파괴된다.
  • 상대 라인이 비어 있으면 몬스터가 상대 LP에 직접 피해를 준다.

마법 카드

구분동작
앞면 마법라인에 배치한 뒤 몇 턴 후 효과가 발동된다. 발동 전에 파괴되면 효과가 사라진다.
뒷면 마법뒤집어 배치하고, 조건이 달성되면 공개되며 효과가 발동된다.

마법은 전투력 상승, 상대 몬스터 제거, 상대 마법 제거를 중심으로 라인의 규칙을 바꾸는 장치처럼 설계한다.

화면 구성

  • 기준 해상도는 1600 x 900이다.
  • 실행파일 창은 모니터 작업영역에 맞춰 16:9 비율로 자동 축소된다.
  • 예시 이미지처럼 좌측에는 손패, 중앙에는 3라인 전장, 우측에는 LP와 커밋 버튼을 배치한다.
  • 덱 빌더도 가로 화면에 맞춰 카드 아카이브와 현재 덱 패널을 좌우로 분리한다.
  • 카드 기본 크기: 132 x 188.
  • 라인 기본 크기: 198 x 286.
  • 손패 패널 높이: 318.
  • 좌측 손패 패널 크기: 440 x 760.
  • 손패 카드 표시 배율: 150%.
  • 전투 라인 패널: 850 x 96.
  • 라인 클릭 히트 영역: 222 x 296.
예시 이미지의 가로 보드 구도를 참고해 1600 x 900 레이아웃으로 전환했다. 좌측 손패 카드는 필드 카드 대비 150%로 표시하며, 현재 소환 또는 배치가 가능한 카드는 초록색 글로우로 표시한다. 손패가 묻히지 않도록 좌측 2열 그리드로 고정했다.
