YugiohLegend 기획서

업데이트: 2026-06-17 / 현재 버전: v0.1.1

3라인과 4턴 구조로 빠른 판단과 명확한 결과 피드백을 제공하는 전술 카드 게임.

1600 x 900 3라인 4턴 12장 덱

문제 정의

카드 배틀 게임을 좋아하는 플레이어가 짧은 시간 안에 라인 선택, 소환, 마법 배치를 판단하고 싶지만, 기존 룰은 준비 단계와 전투 결과를 한눈에 이해하기 어렵다.

주 페르소나

  • 민준, 29세 직장인. 출퇴근 후 5~10분 단위로 카드 게임을 즐긴다.
  • 복잡한 덱 빌딩보다 빠른 전투 판단과 명확한 카드 효과 피드백을 선호한다.

핵심 루프

유저가 손패에서 카드 선택 → 라인에 소환/마법 배치 → COMMIT으로 턴 확정 → 전투와 마법 효과 확인 → 다음 턴 손패와 필드 상황으로 다시 카드 선택.

MVP 가설

  • 3개 라인만 사용하면 초보자도 한 턴 안에 공격/방어 의도를 파악할 수 있다.
  • 4턴 제한은 한 판의 템포를 짧게 유지해 반복 플레이를 유도한다.
  • 상급 몬스터는 필드 몬스터를 직접 제물로 선택해 교체 소환할 때 조작 의도가 가장 명확하다.

레퍼런스 분석

레퍼런스핵심 행동 단계 수적용 교훈
Yu-Gi-Oh! Tribute Summon3단계상급 소환은 제물 선택이 조작의 핵심이므로 자동 처리만으로는 손맛이 약하다.
Marvel Snap lane play3단계라인 수를 줄이면 카드 선택 결과를 빠르게 읽을 수 있다.
Hearthstone minion combat3단계결과 피드백은 턴 확정 후 묶어서 보여주는 편이 이해하기 쉽다.

성공 KPI

  • 첫 판 완료율 80% 이상
  • 튜토리얼 없이 첫 소환 성공률 90% 이상
  • 상급 소환 시도 후 성공/실패 원인 이해율 85% 이상
  • 평균 한 판 플레이 시간 7분 이하

기본 규칙

  • 시작 LP는 4000, 시작 손패는 4장, 덱은 12장이다.
  • 총 4턴으로 진행하며 1턴은 배치 전용 턴이다.
  • 1턴은 1라인, 2턴은 2라인, 3턴 이후는 3라인을 사용한다.
  • 매 턴 시작 시 지연 중인 앞면 마법을 먼저 해결한 뒤 드로우와 행동 선택을 진행한다.

제물 소환

  • 상급 몬스터 카드를 선택하면 제물 후보 라인이 하이라이트된다.
  • 제물 몬스터가 있는 라인을 클릭하면 그 몬스터를 제물로 바치고 같은 라인에 상급 몬스터를 소환 큐에 넣는다.
  • 빈 라인을 선택해도 소환할 수 있으며, 이 경우 필요한 제물은 다른 아군 몬스터 중 전투력이 낮은 순서로 자동 선택된다.
  • 제물이 2장 이상 필요한 경우 클릭한 라인의 몬스터를 먼저 제물로 사용하고 부족한 수량은 자동 선택한다.
  • COMMIT 후 서버는 제물 수, 목표 라인 해금 여부, 손패 카드 여부를 검증하고 성공 시 제물 몬스터를 제거한 뒤 상급 몬스터를 배치한다.

마법 카드

  • 함정은 별도 카드 종류로 쓰지 않고 앞면 마법과 뒷면 마법으로 통합한다.
  • 앞면 마법은 공개 배치되고 다음 턴 시작 시 효과가 발동한다.
  • 뒷면 마법은 조건을 만족하면 공개되며 효과를 발동한다.
  • Second Gate는 다음 턴 소환 가능 횟수를 1회 추가한다.
  • Overextend Collapse는 상대가 이번 턴 몬스터를 2회 이상 성공적으로 소환하면 필드 위 모든 몬스터를 제거한다.

UI/UX

  • 손패, 3라인 전장, HUD, 덱 카운터, 상태 가이드, COMMIT 버튼을 한 화면에서 확인한다.
  • 소환 또는 배치 가능한 카드는 금색 글로우로 표시한다.
  • 상급 몬스터 선택 시 제물 후보 라인과 실제 소환 가능 라인을 함께 안내한다.
  • 현재 버전은 마나 비용과 마나 게이지를 사용하지 않는다.
