YugiohLegend 기획서

3라인 자동 전투 카드 게임. 플레이어는 덱에서 뽑은 몬스터와 마법을 라인에 배치하고, 공개 후 각 라인에서 자동 전투를 해결한다.

1600 x 900 가로 화면 3라인 전장 4턴 매치 앞면/뒷면 마법 덱 구성 과정 추가

핵심 규칙

  • 시작 LP는 4000이다.
  • 시작 손패는 4장이며 매 턴 1장을 드로우한다.
  • 총 4턴으로 진행하고, 1턴은 공격하지 않는 배치 전용 턴이다.
  • 2턴부터 공개 후 라인별 자동 전투가 발생한다.
  • 4턴 종료 후 LP가 높은 쪽이 승리한다.

라인 해금

턴사용 가능한 라인
1턴중앙 1라인
2턴좌측 + 중앙 2라인
3턴좌측 + 중앙 + 우측 3라인
4턴전체 3라인

카드 구분

구분설명
FREE제물 없이 일반 소환 가능한 몬스터
TRIBUTE이미 필드에 소환된 아군 몬스터를 바쳐야 소환 가능한 몬스터
STRIKER전투력이 높고 생명력이 낮은 기본 몬스터
GUARDIAN생명력이 높고 전투력이 낮은 기본 몬스터
UTILITY전투력과 생명력은 낮지만 이동 또는 특수효과가 있는 기본 몬스터
BRUISER / ALLY+ / FIELD+ / MOBILE / SCALE강력한 제물 몬스터의 역할 구분

제물 소환은 손패가 아니라 이미 필드에 소환된 아군 몬스터만 바칠 수 있다. 제물이 필요한 몬스터는 먼저 소환할 라인에 내려놓고, 이후 필드에 있는 아군 몬스터 중 필요한 수만큼 직접 클릭해 제물로 선택한다. 필요한 제물을 모두 고르기 전에는 COMMIT이 비활성화된다.

마법 카드

앞면 마법

라인에 공개 배치되며 몇 턴 뒤 효과가 발동한다. 발동 전에 제거되면 효과가 사라진다.

뒷면 마법

뒤집어 배치하고 조건이 달성되면 공개되며 강한 효과를 발동한다.

주요 효과

전투력 상승, 상대 몬스터 제거, 상대 마법 제거를 중심으로 라인의 규칙을 바꾼다.

덱 구성 과정

  1. Deck Builder에서 카드 아카이브의 카드를 선택한다.
  2. 덱은 반드시 12장을 모두 채워야 한다.
  3. 현재 덱의 몬스터, 기본 몬스터, 제물 몬스터, 앞면 마법, 뒷면 마법 수를 즉시 확인한다.
  4. 현재 덱은 텍스트 목록이 아니라 12개의 슬롯으로 표시한다.
  5. 카드 아카이브의 카드는 클릭으로 첫 빈 슬롯에 넣거나, 드래그해서 원하는 빈 슬롯에 직접 넣을 수 있다.
  6. 슬롯에 들어간 카드는 카드 미니 프리뷰로 표시하고, 클릭하면 제거된다.
  7. AUTO FILL로 초보자용 추천 덱을 슬롯에 자동 구성할 수 있다.
  8. SAVE DECK은 저장만 수행하고, SAVE AND DUEL은 저장 후 대전을 시작한다.
  9. 메인 메뉴의 SOLO DUEL은 유효한 저장 덱이 있으면 그 덱으로 시작하고, 없으면 Deck Builder로 이동한다.

전투 규칙

  • 같은 라인에 양측 몬스터가 있으면 전투력 차이만큼 약한 쪽 몬스터의 생명력을 잃는다.
  • 생명력이 0 이하가 된 몬스터는 파괴된다.
  • 상대 라인이 비어 있으면 몬스터가 상대 LP에 직접 피해를 준다.

최신 UI 구성

  • 가로형 전장 레이아웃을 사용한다.
  • 소환 또는 배치 가능한 카드는 녹색 글로우로 표시한다.
  • 카드에는 종류, 사용 방식, ATK/HP, 제물 필요 여부를 한눈에 보이게 표시한다.
  • 버튼, HUD, 손패 보관대, 라인 프레임은 전용 텍스처를 사용한다.
  • 상단 가이드 HUD는 짧은 2줄 안내로 고정해 프레임 밖으로 문구가 잘리지 않게 한다.
  • 필드에 소환된 말은 라인 프레임보다 아주 살짝 크게 표시해 판 밖으로 조금 돌출되는 연출을 준다.
  • 현재 버전에는 마나 비용/마나 게이지 개념을 사용하지 않는다.
  • 덱 편성 화면의 카드 아카이브와 슬롯 카드 프리뷰는 이전보다 크게 표시해 카드 정보를 더 쉽게 읽게 한다.
