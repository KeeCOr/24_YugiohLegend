YugiohLegend 기획서

3개 라인, 4턴, 동시 행동 공개를 중심으로 한 간단한 카드 대전 게임. 1턴은 공격 없이 필드를 준비하고, 2턴부터 자동 전투가 해결된다.

버전 v0.1.0 최종 갱신 2026-05-18 화면 900 x 1600 플랫폼 Windows / Electron

핵심 규칙

  • 각 플레이어는 3개 라인을 사용한다.
  • 시작 LP는 4000이다.
  • 시작 손패는 4장이고, 매턴 1장을 드로우한다.
  • 한 턴에 몬스터는 1장만 소환할 수 있다.
  • 마법은 여러 장 사용할 수 있지만, 해금된 라인에 먼저 배치해야 한다.
  • 1턴은 셋업 턴이다. 소환과 마법 배치만 처리하고 공격은 하지 않는다.
  • 2~4턴은 행동 공개 후 자동 전투를 처리한다.
  • 4턴 종료 후 마지막 전투를 처리하고, LP가 높은 쪽이 승리한다.

라인 해금

턴사용 가능 라인의도
1턴1번 라인공격 없이 필드의 첫 축을 만든다.
2턴1~2번 라인전투가 시작되고 선택지가 늘어난다.
3턴1~3번 라인 전체전체 라인이 열려 본격적인 라인 싸움이 된다.
4턴1~3번 라인 전체마지막 전투와 승패 결정에 집중한다.

서버 검증, AI 선택, 클라이언트 클릭 판정, 잠금 오버레이는 모두 같은 3라인 해금 규칙을 사용한다.

카드 사용 조건과 등급

카드는 손패에서 바로 구분할 수 있도록 사용 조건과 별 등급을 표시한다.

즉시 사용
제물 없이 바로 낼 수 있는 기본 몬스터. 카드에는 FREE 또는 즉시 사용 배지가 표시된다.
제물 필요
강력한 몬스터는 필드의 아군 몬스터를 제물로 바쳐야 한다. 카드에는 TRIBUTE 배지가 표시된다.
별 등급
카드의 체급을 별 개수로 보여준다. 일반 몬스터는 낮은 별, 제물 몬스터는 높은 별을 가진다.

마법 카드

  • 마법은 낸 직후 즉발하지 않고, 라인에 배치된 뒤 지정 턴 수가 지나면 발동한다.
  • 앞면 마법은 공개된 지연 효과다. 예: 전투력 상승, 상대 몬스터 제거, 상대 마법 제거.
  • 뒷면 마법은 숨겨진 조건부 효과다. 예: 공격 무효, 직접 공격 피해 감소.
  • 발동 전에 해당 라인이 공격받으면 배치된 앞면 마법은 파괴되어 효과가 사라진다.
  • 상대에게는 뒷면 마법이 예약 카드로만 보인다.

전투 처리

  • 같은 라인에 양쪽 몬스터가 있으면 전투력 차이만큼 약한 쪽이 피해를 받고 몬스터가 파괴된다.
  • 공격하는 라인에 상대 몬스터가 없으면 직접 공격으로 상대 LP가 감소한다.
  • 양쪽 전투력이 같으면 두 몬스터가 함께 파괴된다.
  • 라인에 배치된 뒷면 마법 조건이 맞으면 전투에 개입한다.

UI 구성

  • 상단에는 상대 LP와 상대 필드를 보여준다.
  • 중앙에는 3개 라인의 전투 구역을 크게 보여준다.
  • 하단에는 내 LP, 커밋 버튼, 손패를 배치한다.
  • 잠긴 라인은 어둡게 표시하고, 해금 턴을 안내한다.
  • 카드는 종류, 사용 조건, 별 등급, ATK/HP가 손패에서도 보이도록 표시한다.
