YugiohLegend 기획서

3라인 자동 전투 카드 게임. 손패에서 카드를 고르고 라인에 배치한 뒤, 공개와 전투를 통해 4턴 안에 LP 우위를 만든다.

1600 x 900 가로 화면 3라인 전장 4턴 매치 앞면/뒷면 마법 카드 아트 적용

핵심 규칙

  • 시작 LP는 4000이다.
  • 시작 손패는 4장이고 매 턴 1장을 드로우한다.
  • 총 4턴으로 진행하며, 1턴은 공격하지 않는 배치 전용 턴이다.
  • 2턴부터 공개 후 라인별 자동 전투가 발생한다.
  • 4턴 종료 후 LP가 높은 쪽이 승리한다.

라인 해금

턴사용 가능한 라인
1턴중앙 1라인
2턴왼쪽 + 중앙 2라인
3턴왼쪽 + 중앙 + 오른쪽 3라인
4턴전체 3라인

카드 구분

구분설명
FREE제물 없이 일반 소환 가능
TRIBUTE제물이 필요한 상급 소환
STRIKER전투력은 높지만 생명력이 낮은 기본 몬스터
GUARDIAN생명력은 높지만 전투력이 낮은 기본 몬스터
UTILITY능력치가 낮지만 이동 또는 특수효과가 있는 기본 몬스터
BRUISER / ALLY+ / FIELD+ / MOBILE / SCALE제물을 요구하며 강한 능력치, 아군 강화, 필드 강화, 이동성, 제물 비례 성장 중 하나를 담당

전투와 마법

전투

같은 라인에 양측 몬스터가 있으면 전투력 차이만큼 약한 쪽이 생명력을 잃고, 생명력이 0 이하가 되면 파괴된다.

앞면 마법

라인에 공개 배치되며 몇 턴 뒤 효과가 발동한다. 발동 전에 제거되면 효과가 사라진다.

뒷면 마법

뒤집어 배치하고 조건을 만족하면 공개되며 강한 효과를 발동한다. 별도 함정 카드는 두지 않는다.

최신 UI 구성

  • 왼쪽 손패, 중앙 3라인 전장, 오른쪽 상대/플레이어 HUD와 커밋 버튼으로 구성한다.
  • 손패는 세로 팬 형태로 펼쳐 카드 겹침을 줄이고, 플레이 가능한 카드는 글로우 처리한다.
  • 전장 상단에 게임명, 턴, 단계, 입력 가이드를 표시한다.
  • 오른쪽 HUD에 생성 카드 이미지를 활용한 상대/플레이어 초상과 덱 카운터를 추가한다.
  • 앞면/뒷면 마법 슬롯에도 해당 카드 이미지를 작은 아이콘으로 표시한다.

카드 아트 적용

  • 현재 카드 구성 18종에 맞춰 AI 카드 일러스트를 생성했다.
  • 원본 아틀라스: client/public/assets/cards/card_art_atlas.png
  • 카드별 경로: client/public/assets/cards/{cardId}.png
  • BootScene에서 모든 카드 이미지를 로드하고 CardSprite가 카드 ID에 맞는 이미지를 표시한다.
  • 주요 HUD 초상과 마법 슬롯 아이콘에도 같은 이미지를 재사용한다.
