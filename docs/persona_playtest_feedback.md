## 플레이 전 기대

카드 소환과 LP 싸움으로 승부를 내는 듀얼 카드 게임.

이 페르소나는 이 프로젝트에서 **LP를 깎아내며 마지막 공격으로 승부가 끝나는 카타르시스**을 기대한다.

## 테스트 피드백

- 좋았던 점: LP를 깎아내며 마지막 공격으로 승부가 끝나는 카타르시스
- 헷갈린 점: LP 0 순간의 연출이 약하면 듀얼 전체의 끝맛이 흐려진다.
- 이탈 가능 지점: LP 0 마무리 연출 방향을 확정해 전투의 끝맛을 정해야 한다.
- 1순위 개선 제안: LP 0 직전 경고, 마지막 타격, 승리 선언을 3단 연출로 고정한다.
