YugiohLegend UX 및 튜토리얼 개선 대책

목표는 첫 플레이어가 현재 무엇을 할 수 있고, 왜 할 수 없고, 다음에 무엇을 눌러야 하는지 즉시 이해하게 만드는 것이다.

업데이트: 2026-05-28 대상: 1600 x 900 가로형 UI 상태: 핵심 UX 일부 적용 완료

핵심 문제

목표가 불명확함
처음 화면에서 어떤 카드와 라인을 선택해야 하는지 알기 어렵다.
행동 가능 여부가 약함
글로우만으로는 왜 가능한지, 왜 불가능한지 설명이 부족하다.
입력 실패가 조용함
라인을 먼저 누르거나 COMMIT을 먼저 눌렀을 때 명확한 피드백이 필요하다.
전투 결과 예측이 어려움
ATK/HP 차이, 라인 해금, 마법 지연 발동을 더 눈에 띄게 안내해야 한다.

적용 완료

영역적용 내용효과
손패 사용 가능한 카드는 녹색 글로우, 사용할 수 없는 카드는 어둡게 표시한다. 플레이어가 이번 턴에 낼 수 있는 카드를 바로 구분한다.
손패 hover 카드에 마우스를 올리면 가능한 라인 또는 불가능한 이유를 상태 메시지에 표시한다. 제물 부족, 빈 라인 없음, 이미 소환함 같은 실패 원인을 즉시 이해한다.
라인 안내 카드를 선택하면 실제 배치 가능한 라인만 녹색으로 강조한다. 카드 선택 후 어디를 눌러야 하는지 명확해진다.
COMMIT 배치한 카드가 없으면 버튼을 비활성처럼 표시하고, 클릭 시 먼저 카드를 내라는 메시지를 표시한다. 빈 행동 제출을 줄이고 턴 진행 조건을 이해시킨다.
라인 선입력 카드를 선택하지 않은 상태로 라인을 누르면 손패 카드를 먼저 선택하라고 안내한다. 입력 실패가 버그처럼 느껴지는 문제를 줄인다.

다음 우선순위

  • 전투 전에 예상 피해와 생존 여부를 라인 중앙에 미리 표시한다.
  • 첫 실행 시 3단계 튜토리얼 오버레이를 보여주고, 이후에는 다시 보기 버튼으로 접근하게 한다.
  • 앞면 마법에는 남은 턴 카운트, 뒷면 마법에는 조건 아이콘을 더 크게 표시한다.
  • 제물 소환 카드 선택 시 자동으로 제물 후보 라인을 붉은색 또는 금색으로 표시한다.
가장 중요한 원칙은 “안 되는 행동을 막는 것”보다 “왜 안 되는지 바로 설명하는 것”이다.
